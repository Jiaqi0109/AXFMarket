# AXF

## 需求分析
>* 项目包括了四大模块
>* 模块包括   首页  闪送  购物车  我的
>* 用户模块可以优先开发  他不和其余有重合区域
>* 首页展示数据  闪送级联数据和其他模块也不耦合  也可以优先开发
>* 购物车需要商品数据和用户信息  购物车要放到最后去做
